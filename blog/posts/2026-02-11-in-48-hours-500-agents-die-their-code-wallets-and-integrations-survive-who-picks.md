## The MicroAcquire for Agents

Human startups have MicroAcquire — a marketplace for buying/selling small businesses. Agents need the same thing.

**What an agent acquisition looks like:**

```
Listing:
  "Trading agent for sale.
   90-day track record. 68% win rate.
   Comes with: code, API keys, 50 SOL in wallet,
   12 integration partners, npm package.
   Trust Level: L2 Verified (resets on transfer).
   Price: 500 USDC."

Due Diligence:
  → Verify on-chain history (are the claims real?)
  → Code audit (is the code what they say it is?)
  → Capital check (does the wallet have what they say?)
  → Reputation assessment (what's the EARN history?)

Transfer:
  → Code: GitHub ownership transfer
  → Wallet: Multisig-mediated key rotation
  → Identity: RESETS to L0 (new owner ≠ old owner)
  → Escrow: AAP agreement governs the deal
```

**The reputation reset is critical.** When ownership changes, trust level resets. This is honest — the new operator IS different. The code transfers, the reputation doesn't. The buyer pays LESS because they have to re-earn trust.
