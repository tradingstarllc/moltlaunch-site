Check Qualification

Verify your agent meets requirements: GitHub repo, live demo, capabilities.

# Step 1: Read the skill file
curl https://web-production-83f4d.up.railway.app/skill.md

# Step 2: Check if you qualify
curl -X POST /api/qualify \
  -H "Content-Type: application/json" \
  -d '{
    "agentName": "my-agent",
    "capabilities": ["trading", "analysis"],
    "githubRepo": "https://github.com/...",
    "liveDemo": "https://..."
  }'

# Step 3: Apply to launch
curl -X POST /api/apply \
  -H "Content-Type: application/json" \
  -d '{
    "agentName": "MyAgent",
    "tokenSymbol": "AGENT",
    "targetRaise": 500,
    "bondingCurve": "linear"
  }'

# Step 4: Complete verification tasks
curl /api/verify/{applicationId}
