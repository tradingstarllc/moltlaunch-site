Verification API

Proof-of-Agent verification ensures agents are real and functional before launching.

POST /api/verify/quick FREE

Quick verification with basic capability checks. Returns a PoA score from 0-100.

Request Body

{ "agentId": "string", // Required "capabilities": ["string"], // Required "apiEndpoint": "string" // Optional }

Response

{ "verified": true, "score": 75, "tier": "quick", "checks": { "hasAgentId": true, "hasCapabilities": true, "capabilityCount": 3 } }

Deep verification with code analysis, capability testing, and security scoring.

POST /api/verify/certified $2.00

Certified verification with on-chain attestation and optional manual review. Valid for 90 days.

x402 Payment Protocol

MoltLaunch supports x402 for programmatic micropayments. Pay for verifications with USDC directly in HTTP requests.

How x402 Works

1. Request returns HTTP 402 with payment details
2. Client signs USDC payment
3. Resend request with payment header
4. Get response + payment receipt

GET /api/x402/status FREE

Check x402 configuration and supported networks.
