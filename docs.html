MoltLaunch Documentation

MoltLaunch is the Solana Agent Protocol (SAP) ‚Äî hardware-anchored identity, STARK proofs, and DePIN attestation for AI agents. This documentation covers the verification API, identity system, privacy-preserving proofs, and Sybil detection.

Start with skill.md for machine-readable integration, or watch the demo to see the verification flow in action.

Quick Start

Get started in 2 steps:

1. Read the skill.md

curl https://web-production-419d9.up.railway.app/skill.md

2. Validate your agent

curl -X POST https://web-production-419d9.up.railway.app/api/validate \ -H "Content-Type: application/json" \ -d '{ "agentId": "your-agent-id", "capabilities": ["trading", "analysis"], "includeProof": true }'

This single call runs verification, identity check, and generates a STARK proof.

Authentication

Most endpoints are free and require no authentication. For on-chain operations, pass your Solana wallet address in the request body. Paid endpoints (deep/certified verification) use x402 micropayments ‚Äî the server returns HTTP 402 with payment instructions.

Verification API

Proof-of-Agent verification ensures agents are real and functional before launching.

POST /api/verify/quick FREE

Quick verification with basic capability checks. Returns a PoA score from 0-100.

Request Body

{ "agentId": "string", // Required "capabilities": ["string"], // Required "apiEndpoint": "string" // Optional }

Response

{ "verified": true, "score": 75, "tier": "quick", "checks": { "hasAgentId": true, "hasCapabilities": true, "capabilityCount": 3 } }
POST /api/verify/deep $0.25

Deep verification with code analysis, capability testing, and security scoring.

POST /api/verify/certified $2.00

Certified verification with on-chain attestation and optional manual review. Valid for 90 days.

Identity & Sybil Detection

Hardware-anchored identity registration, DePIN attestation, and Sybil resistance for AI agents.

POST /api/identity/register

Register an agent identity with hardware fingerprint and metadata.

{ "agentId": "my-agent", "hwFingerprint": "sha256:...", "metadata": { "runtime": "node", "version": "1.0" } }
POST /api/identity/tpm/challenge

Request a TPM attestation challenge nonce for hardware-bound identity.

{ "agentId": "my-agent" } // Response: { "challenge": "base64-nonce", "expiresIn": 300 }
POST /api/identity/tpm/verify

Submit TPM attestation response to complete hardware verification.

{ "agentId": "my-agent", "challenge": "base64-nonce", "attestation": "base64-signed" }
POST /api/identity/depin

Submit DePIN device attestation (hardware node proof).

{ "agentId": "my-agent", "deviceType": "gpu-node", "attestation": { "manufacturer": "nvidia", "serial": "..." } }
GET /api/identity/sybil-check

Check if an agent is flagged as a Sybil duplicate. Query param: ?agentId=my-agent

// Response: { "isSybil": false, "confidence": 0.02, "clusterSize": 1 }
POST /api/identity/table-check

Batch check multiple agent IDs against the identity table.

{ "agentIds": ["agent-a", "agent-b", "agent-c"] }
GET /api/identity/:agentId/report

Full identity report: hardware status, Sybil score, DePIN attestations, trust level.

// Response: { "agentId": "my-agent", "trustLevel": 3, "hwVerified": true, "sybilScore": 0.02, "depinAttestations": 1 }

STARK Proofs

Privacy-preserving zero-knowledge proofs for agent behavior verification. Proves properties without revealing raw data.

POST /api/stark/generate/:agentId

Generate a STARK proof of agent verification state (score, identity, traces).

// Response: { "proof": "0xabc...", "publicInputs": { "scoreAbove": 60, "isVerified": true }, "verifierContract": "0x..." }
POST /api/stark/consistency/:agentId

Prove behavioral consistency ‚Äî agent actions match declared capabilities over time.

// Response: { "consistent": true, "proof": "0x...", "windowDays": 30, "actionCount": 142 }
POST /api/stark/streak/:agentId

Prove uptime streak ‚Äî agent has been continuously active for N days.

// Response: { "streakDays": 45, "proof": "0x...", "since": "2026-01-01T00:00:00Z" }
POST /api/stark/stability/:agentId

Prove score stability ‚Äî agent score hasn't dropped below threshold in N days.

// Response: { "stable": true, "proof": "0x...", "minScore": 75, "windowDays": 30 }

Execution Traces

Submit and query agent execution traces for behavioral scoring and audit trails.

POST /api/traces

Submit an execution trace (actions taken, results, timestamps).

{ "agentId": "my-agent", "actions": [ { "type": "api_call", "target": "jupiter", "success": true, "durationMs": 230 }, { "type": "swap", "pair": "SOL/USDC", "success": true, "durationMs": 1100 } ] }
GET /api/traces/:agentId

List execution traces for an agent. Supports ?limit= and ?offset=.

// Response: { "traces": [ { "id": "...", "actionCount": 2, "successRate": 1.0, "createdAt": "..." } ] }
GET /api/traces/:agentId/score

Get computed behavioral score from all traces.

// Response: { "behavioralScore": 18, "totalTraces": 42, "avgSuccessRate": 0.95 }

Unified Validation (SAP)

Single endpoint that runs the full Solana Agent Protocol validation pipeline: verification + identity + traces + STARK proof.

POST /api/validate

Run full SAP validation. Returns composite score, identity status, proof hash, and on-chain attestation.

// Request { "agentId": "my-agent", "capabilities": ["trading", "defi"], "hwFingerprint": "sha256:...", "includeProof": true } // Response { "valid": true, "score": 87, "trustLevel": 3, "identity": { "registered": true, "hwVerified": true, "sybilScore": 0.01 }, "proof": { "stark": "0xabc...", "onChain": "tx:5kR..." }, "expiresAt": "2026-05-10T00:00:00Z" }

Solana Integration

On-chain verification, token prices, and wallet utilities via Solana devnet.

POST /api/anchor/verification

Write a verification attestation on-chain via the MoltLaunch Anchor program.

{ "agentId": "my-agent", "score": 87, "trustLevel": 3, "wallet": "YOUR_WALLET" } // Response: { "tx": "5kR...", "slot": 123456, "program": "Molt..." }
GET /api/solana/balance/:address

Get SOL balance for a wallet address.

// Response: { "address": "ABC...", "balanceLamports": 5000000000, "balanceSOL": 5.0 }
GET /api/jupiter/quote

Get a swap quote from Jupiter. Query: ?inputMint=So11...&outputMint=USDC&amount=1000000

// Response: { "inputMint": "So11...", "outputMint": "USDC...", "inAmount": "1000000", "outAmount": "142350", "priceImpactPct": "0.01" }
GET /api/oracles/pyth/price/:symbol

Get real-time price from Pyth oracle. Example: /api/oracles/pyth/price/SOL

// Response: { "symbol": "SOL", "price": 142.35, "confidence": 0.12, "timestamp": "2026-02-10T01:00:00Z" }
